# Profile Image Setup

To add your profile image to the portfolio:

1. Save your profile image as `profile.jpg`
2. Place it in the `public` folder of your project:
   ```
   portfolio-v1/
   └── public/
       └── profile.jpg
   ```

The Hero section will automatically display your image. If the image is not found, it will fall back to a generated avatar.

## Image Requirements
- **Format**: JPG, PNG, or WebP
- **Recommended size**: 512x512 pixels or larger
- **Aspect ratio**: Square (1:1) works best for circular display
